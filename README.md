# CyberShield Password Analyzer

A cybersecurity inspired web tool that evaluates password strength based on real world security principles.

## About the Project
CyberShield Password Analyzer checks how strong a password is by analyzing:
- Length of password
- Use of uppercase letters
- Use of numbers
- Use of special characters
- Common weak passwords
It then gives a security score out of 100 and a risk level (Weak / Medium / Strong).

## Why I built this
I wanted to understand how websites evaluate password security and learn how real-world cybersecurity systems reduce weak password usage.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Password strength scoring system
- Visual progress bar
- Risk classification (High / Medium / Secure)
- Detection of common weak passwords
