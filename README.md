# Password Attack Simulator

A cybersecurity learning tool that simulates how different attack methods (like brute force and dictionary attacks) affect password security.

## Project Overview

This project demonstrates how passwords can be analyzed from an attacker's perspective. Instead of only checking password strength, it simulates how long it might take to crack a password based on its complexity.

It helps users understand real world cybersecurity risks in an interactive way.

## Features

- Password strength scoring system (0–100)
- Attack type simulation:
  - Dictionary Attack
  - Brute Force Attack
  - Advanced Brute Force Simulation
- Estimated password crack time (simulated)
- Security feedback and recommendations
- Detection of weak/common passwords

## How It Works

The simulator analyzes:
- Password length
- Use of uppercase letters
- Use of numbers
- Use of special characters
- Common weak passwords

Based on these factors, it calculates:
- Security score
- Attack difficulty level
- Estimated crack time

## Why I Built This

I wanted to understand how real cybersecurity systems evaluate password strength from both a defensive and offensive perspective. This project helped me learn how attackers think and how password entropy affects security.
